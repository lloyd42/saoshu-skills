# 规则总表

## A. 后宫文边界
- 男主必须开后宫。
- 女主必须全为女性。
- 单女主、伪后宫、主角娘化后单线恋爱，均不纳入本规则。

## B. 六大雷点（命中即高优先级劝退）
1. 绿帽：女主跟男主后与其他男性发生关系（含被强迫）；含主角被女性绿。
2. 死女：女主/准女主死亡且无复活可能；猎奇死亡即使复活也按雷；主角独长生女主老死也按雷。
3. 送女：男主把有明显情感交集且戏份足的女主/准女主送给配角。
4. 背叛：女主/准女主在行为或思想上背叛男主，或彻底离开转投他人。
5. wrq：女主与多名男性发生关系。
6. 龟作：作者知雷写雷，如前世/原线大规模刻意写fc/wrq等。

## C. 郁闷点与最低可抗防御

### C1 前世雷（重生/穿书常见）
- 严重：前世强迫、wrq、背叛主角 -> 重甲以上
- 中等：前世fc/嫁人、严重亵女 -> 重甲
- 轻微：前世背刺、中等亵女 -> 布甲

### C2 px/fc/非初
- px（多次嫁人fc）-> 重甲以上
- 第一女主fc（非mn特例）-> 重甲
- 无意义fc -> 布甲
- 嫁人fc -> 轻甲
- mn里的fc -> 低防
- 无意义非初 -> 轻甲
- 曹贼非初（无肢体关系）-> 低防

### C3 亵女
- 严重（体液接触敏感部位/脱光摸敏感部位）-> 重甲以上
- 中上（被看光/夫前目犯）-> 重甲
- 中等（泳衣被看/骚扰摸身等）-> 布甲
- 轻微（礼节握手/现代穿着等）-> 轻甲
- 超轻微（言语调戏）-> 低防

### C4 关系与结构类
- 漏女：<3个 -> 轻甲；3-7个 -> 布甲；漏成筛子 -> 重甲
- 拒女：重甲；伪拒女 -> 布甲
- 惧女：严重 -> 重甲；中等 -> 布甲；轻微 -> 轻甲
- 虐主：极度 -> 重甲以上；严重 -> 重甲；中等 -> 布甲；轻微 -> 轻甲；超轻微 -> 低防

### C5 特殊题材类
- 百合：严重 -> 重甲以上；中等 -> 布甲；轻微 -> 轻甲；超轻微 -> 低防
- 面具流：严重 -> 重甲；中等 -> 布甲；轻微 -> 轻甲；超轻微 -> 低防
- 分身流：严重（有意识男分身/操控发生关系）-> 重甲；中等 -> 布甲；轻微 -> 轻甲；女性分身 -> 低防
- 生孩子：严重（儿子开后宫/老登转世）-> 重甲以上；中上 -> 重甲；中等 -> 布甲；轻微 -> 轻甲；超轻微 -> 低防
- 接盘：严重 -> 重甲；中等 -> 布甲；轻微 -> 轻甲
- 兄弟情节：严重 -> 重甲；中等 -> 布甲；轻微 -> 轻甲
- 神雕：布甲/轻甲

### C6 其他常见郁闷
- 药老 -> 布甲
- 宠物（高等智慧坐骑）-> 布甲；普通战斗宠物 -> 低防/轻甲
- 被系统控制（有害）-> 布甲；（有利）-> 轻甲
- 圣母、虐心、nc、拉皮条 -> 轻甲/低防
- 魂穿、配角开后宫、同人凑CP、擦边、无意义郁闷 -> 低防

## D. 证据级别标签（强制）
- 已确认：有明确剧情或用户给出明确片段。
- 高概率：多处线索指向，但缺关键实锤。
- 待补证：信息不足，不能判定；`未知待证`仅作为兼容旧字段的别名。

## E. 结论判定
- 命中雷点：
  - 神防之上：可看（雷点提示）
  - 神防：除绿帽外可看但重警告；绿帽劝退
  - 其余防御：劝退
- 无雷时：按最高郁闷等级匹配可抗防御给出可看/慎入。
